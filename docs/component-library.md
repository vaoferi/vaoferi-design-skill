# Component Library

Компонентна бібліотека - це набір повторюваних UI-елементів, які можна комбінувати у різних макетах.

## Мета

Зменшити дублікати, зберегти візуальну консистентність і швидко будувати нові сторінки.

## Як працювати з бібліотекою

1. Перед створенням нового елемента перевір існуючі компоненти.
2. Якщо потрібен новий компонент - поясни, чому існуючого недостатньо.
3. Запропонуй назву, варіанти, поведінку.
4. Отримай затвердження перед додаванням.

## Базові компоненти (приклади)

- Button (primary/secondary/danger)
- Input / Textarea / Select
- Card (заголовок, контент, дії)
- Badge / Tag
- Alert / Toast
- Modal / Drawer
- Tabs / Accordion
- Skeleton loader

## Правила роботи

- Компонент описується через API: props, events, variants, states.
- Не дублюй логіку, яка вже є в схожому компоненті.
- Document new components so the next AI knows how to use them.
