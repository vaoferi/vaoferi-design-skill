# Design Tokens

Design tokens - це іменовані значення для кольору, типографіки, відступів, радіусів, тіней, меж і розмірів.

## Чому це важливо

Токени зберігають консистентність і спрощують підтримку, теми та адаптивність.

## Приклад патерну

```code
--color-primary: #0f172a;
--color-accent: #3b82f6;
--radius-md: 8px;
--space-4: 16px;
--font-body: "Inter", system-ui, sans-serif;
```

## Рекомендації

- Не вигадуй CSS-значення вручну, якщо вже є токен.
- Для нових токенів запитуй назву, контекст використання і варіанти.
- Не змішуй продуктові значення і технічні утиліти в одному рівні.
